Aprender Git e GitHub 

 

Cadastro do nome: git config --global user.name claubis 

 

Cadastrar o e-mail: git config --global user.email c...



Confirmar se deu certo o nome: git config user.mail 

Confirmar se deu certo o e-mail: git config user.email 

 

Para criar uma pasta nova: mkdir FIAP-git 

 

Confirmar se e mudar para pasta que desejo: cd FIAP-GIT (nome da pasta) 

Iniciar a pasta: git init 

Mostrar os arquivos presentes na pasta: ls –la 

Conferir o status dos documentos: git status 

Exemplo: 

On branch master – Mostra a Branch principal 

No commits yet – Mostra de houve commits 

Untracked files: 

  (use "git add <file>..." to include in what will be committed) 

        desktop.ini 

        primeiro.txt 

  

nothing added to commit but untracked files present (use "git add" to track) 

 

Adicionar o na lista os arquivos que preciso comitar: git add primeiro.txt (Nome e extensão do arquivo) 

Git status: 

 

On branch master 

No commits yet 

Changes to be committed: 

  (use "git rm --cached <file>..." to unstage) 

        new file:   primeiro.txt -- Aqui mostra que foi inserido um arquivo para commit 

  

Untracked files: 

  (use "git add <file>..." to include in what will be committed) 

        Desktop.ini 

Para comitar os arquivos: git commit -m "primeiro.txt" 

Para saber os comits que fizemos: git log

Para conectar no GitHub 

git remote add origin https://github.com/Claubis/FIAP.git 

Saber qual origem estamos: git remote 

Para enviar o arquivo: git push -u origin master (Colocar a branch) 

Para confirmar em qual branch estou: git branch
